# Real-Time Database Update System

## Overview

This project demonstrates a real-time database update system using PostgreSQL, Node.js, and Socket.IO.

Whenever changes occur in the `orders` table (INSERT, UPDATE, DELETE), connected clients receive live updates instantly without polling.

---

# Technologies Used

- Node.js
- Express.js
- PostgreSQL
- Socket.IO
- HTML/CSS/JavaScript

---

# Features

- Real-time database updates
- Dashboard statistics
- Live activity feed
- Search/filter
- Dark mode
- Charts visualization
- Status badges
- Connection status indicator
